# SpeechGenderRecognition

SpeechGenderRecognition


## Requirements

- Python3

**Libraries**
- scikit-image
- pandas
- numpy
- matplotlib

You can install this requirements with
```
$ pip3 install -r requirements.txt
```

## Usage

To run the classifiers use:

```
python3 main.py -r
```

To run the classifier with a new sample use:
```
python3 main.py -i path/file.csv
```
